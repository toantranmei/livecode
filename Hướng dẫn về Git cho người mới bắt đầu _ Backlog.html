Hướng dẫn về Git cho người mới bắt đầu -- Sử dụng hệ thống quản lý phiên bản của Git cách chuyên nghiệp

Chào mừng bạn đến với trang hướng dẫn Git dành cho người mới bắt đầu sử dụng. Tại đây, chúng tôi sẽ hướng dẫn bạn dùng Git để quản lý các phiên bản một cách thành thạo. Có 3 phần trong hướng dẫn này. Nếu Git hoàn toàn mới mẻ với bạn, bạn có thể bắt đầu khám phá từ phần Nhập môn. Người dùng đã từng sử dụng Git thì có thể chuyển đến phần Phát triển để tìm hiểu sâu hơn về ứng dụng này. Phần thứ ba là phần Tài liệu tham khảo nhằm hỗ trợ cho các trường hợp cụ thể.
